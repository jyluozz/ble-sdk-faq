# BLE SDK FAQ

BLE 蓝牙 SDK 常见问题解答系统，面向客户开发者自助排查问题的参考手册，同时也是内部团队知识沉淀的工具。

## 功能特性

- **模块化分类** — 覆盖连接配对、数据传输、低功耗、OTA升级、异常处理五大模块
- **本地存储** — 数据存储于浏览器 localStorage，开箱即用，无后端依赖
- **实时搜索** — 支持问题标题和答案内容的全文搜索
- **快速添加** — 内置 FAQ 添加表单，便于团队持续沉淀
- **代码高亮** — 支持 C/嵌入式代码示例展示

## 技术栈

- 纯原生 HTML/CSS/JS，无框架依赖
- localStorage 前端持久化
- 响应式布局，适配桌面端与移动端

## 快速开始

直接在浏览器中打开 `index.html` 即可使用。

## 内置 FAQ 模块

| 模块 | 图标 | 说明 |
|------|------|------|
| 连接配对 | 📡 | 配对失败、多设备连接、信号强度等问题 |
| 数据传输 | 📤 | MTU配置、小包传输优化、延迟排查 |
| 低功耗 | 🔋 | 待机功耗优化策略 |
| OTA升级 | 🔄 | 升级失败处理与排查 |
| 异常处理 | ⚠️ | 错误码解读与异常恢复 |
