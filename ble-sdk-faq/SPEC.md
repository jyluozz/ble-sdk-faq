**版本：** v1.0.0
**日期：** 2026-05-17
**状态：** 已确认

---

## 1. 概述

### 目的
为BLE SDK生成一份HTML格式的FAQ文档，供客户开发参考，同时也是内部团队知识沉淀的工具。

### 定位
- **对外**：客户开发者自助排查问题的参考手册
- **对内**：内部团队记录和同步FAQ的共享文档

### 技术选型
- **存储**：localStorage（前端本地存储）
- **部署**：HTML文件放在SDK中，同步到Gitee
- **依赖**：纯原生HTML/CSS/JS，无后端依赖

## 4. 数据结构

### FAQ对象

```javascript
{
  id: "faq_001",           // 唯一ID（时间戳+随机数）
  module: "connection",    // 所属模块ID
  question: "配对失败如何排查？",  // 问题标题
  answer: "详细解答...",    // 解答内容
  code: "// 示例代码",       // 代码示例（可选）
  tags: ["新增"],           // 标签：新增/热门/常见
  date: "2026-05-17",      // 添加日期
  author: "员工姓名"        // 添加人（可选）
}
```

### localStorage Key

- `ble_faq_data` — FAQ数据列表（JSON数组）
- `ble_faq_modules` — 模块配置（JSON对象）

## 5. 交互功能

| 功能 | 实现方式 |
|------|----------|
| 搜索 | 实时过滤所有FAQ问题标题 |
| 按模块查看 | 点击模块卡片跳转详情页 |
| 查看单条FAQ | 点击首页FAQ条目跳转详情页锚点定位 |
| 添加FAQ | 弹窗表单，提交后存入localStorage |
| 返回首页 | 面包屑导航 |

## 6. 样式规范

### 视觉风格
- **主题**：极简白底 + 模块化色块（类似Notion/GitHub Docs风格）
- **字体**：系统默认字体栈，支持中英文
- **配色**：
  - 背景：#ffffff（白）
  - 卡片背景：#f6f8fa（浅灰）
  - 文字：#24292f（深灰）
  - 强调：#0969da（蓝色链接）
  - 标签：#ddf4ff（浅蓝背景）

### 布局
- 最大宽度：900px（居中）
- 卡片内边距：1.5rem
- 模块间距：1.5rem
- 响应式：移动端单列，桌面端卡片网格
